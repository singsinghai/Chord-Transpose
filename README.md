# Chord Transpose
An app to transpose with chords on lyrics agreeing the required format. Deploy link: https://share.streamlit.io/singsinghai/chord-transpose/deploy.py

### Usage
Paste lyrics of a song into the input, with the chords indicated inside the format `[{chord}]`. This app will detect all the chords and let you perform tranpose up (#) or tranpose down (b). This is aimed to help musicians with self-draft lyrics scores who can't have prebuilt transpose function.
